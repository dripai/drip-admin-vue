# 后端项目实施清单

本清单用于指导 AI 按后端提示词文档开发项目，并在开发过程中持续更新状态。

状态只允许使用：
- 待开始
- 进行中
- 已完成

执行规则：
- 每次开发前必须先读取本清单。
- 每次只推进少量明确任务，优先完成 P0。
- 功能未完成或未验证，不允许标记为已完成。
- 每次开发结束前必须更新本清单状态。
- 每次提交前必须检查本清单是否同步。

## 0. 项目初始化

| 状态 | 任务 | 验证标准 |
|---|---|---|
| 待开始 | 创建 Spring Boot 后端项目骨架 | 项目可以正常启动 |
| 待开始 | 配置 JDK 25 | 构建工具和运行环境使用 JDK 25 |
| 待开始 | 配置 Spring Boot 4.1.x | 依赖版本明确且项目可启动 |
| 待开始 | 配置 MySQL 8.0 | 后端和 AI 开发流程可以连接 `localhost:3007/drip-manager`，用户名和密码均为 `root` |
| 待开始 | 配置 Redis | 后端可以连接 Redis |
| 待开始 | 配置 Flyway | 可通过迁移脚本自动创建和管理数据库表 |
| 待开始 | 配置 Sa-Token | 登录认证能力可用 |
| 待开始 | 配置 MyBatis-Plus | Mapper 和分页能力可用 |
| 待开始 | 配置 OpenAPI / Swagger | 可以访问接口文档 |
